# Customer Retention Analysis Dashboard

## Project Overview
This project analyzes customer churn and retention using Python and Power BI.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Power BI

## Dataset
Telco Customer Churn Dataset

## Key Insights
- Month-to-month customers have the highest churn rate.
- Two-year contract customers show the lowest churn.
- High-value customers are more likely to be retained.
- Average monthly charge: 64.80
- Average tenure: 32.42 months

## Dashboard KPIs
- Total Customers
- Average Monthly Charge
- Average Tenure
- Total Revenue
